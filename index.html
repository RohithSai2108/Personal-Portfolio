Hand Written Digit Classification

I built an Artificial Neural Network (ANN) model for handwritten digit classification using the MNIST dataset. The model consists of an input layer (784 neurons), hidden layers with ReLU activation, and an output layer with 10 neurons using softmax activation. It was trained using the Adam optimizer and categorical cross-entropy loss. After training, the model achieved high accuracy in classifying digits (0-9). Finally, it was evaluated using metrics like accuracy, precision, and confusion matrix for performance analysis.
